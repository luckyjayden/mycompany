# LuckyJayden 회사홈페이지 — 요구사항 명세서

> 버전: 1.0  
> 작성일: 2026-06-16  
> 원본: spec-origin.md 기반

---

## 1. 프로젝트 목표

기업형 홈페이지를 구축한다.  
방문자에게 회사·제품을 소개하고, 온라인 문의·커뮤니티 게시판·회원관리·관리자 운영 기능을 통합 제공한다.

---

## 2. 제공 기능 범위

| # | 기능 영역 | 설명 |
|---|-----------|------|
| 1 | 회사소개 | 메인 홈페이지에서 회사 개요·수치·특징 제공 |
| 2 | 제품소개 | 제품 목록 및 상세 페이지 |
| 3 | 온라인문의 | 비회원·회원 모두 문의 접수, 관리자 답변 |
| 4 | 커뮤니티 게시판 | 공지·자유·Q&A 게시판 CRUD |
| 5 | 회원관리 | 가입·로그인·프로필 수정·비밀번호 재설정 |
| 6 | 관리자 페이지 | 회원·문의·제품 관리 대시보드 |

---

## 3. 요구사항 목록

### REQ-001 메인 홈페이지 (회사소개)

**분류:** 화면 | **우선순위:** 상

회사를 소개하는 메인 홈페이지를 제공한다.

- 히어로 배너: 슬로건 + 제품/문의 이동 버튼
- 핵심 수치: 고객사 수, 제품 수, 연력, 고객 만족도
- 핵심 특징 카드 3개 (제품군·기술팀·고객지원)
- 회사 소개 텍스트 + 인증·파트너십 체크리스트
- 문의 유도 CTA 섹션
- 모바일·데스크탑 반응형 레이아웃

---

### REQ-002 회원 가입

**분류:** 회원관리 | **우선순위:** 상

사용자는 이름·이메일·비밀번호로 회원 가입을 할 수 있다.

- 비밀번호 6자 이상 필수
- 비밀번호 확인 입력으로 오타 방지
- 이메일 중복 가입 시 오류 메시지 표시
- 가입 완료 즉시 로그인 상태로 메인 페이지 이동

---

### REQ-003 로그인 / 로그아웃

**분류:** 회원관리 | **우선순위:** 상

이메일·비밀번호로 로그인하고, 세션을 종료할 수 있다.

- 로그인 실패 시 "이메일 또는 비밀번호가 올바르지 않습니다." 안내
- 페이지 새로고침 후에도 세션 유지
- 헤더의 사용자 메뉴에서 로그아웃 후 메인 페이지 이동

---

### REQ-004 비밀번호 재설정

**분류:** 회원관리 | **우선순위:** 중

가입된 이메일로 비밀번호 재설정 링크를 발송한다.

- 이메일 입력 → 발송 완료 안내 화면으로 전환
- 링크 클릭 시 새 비밀번호 설정 페이지로 이동

---

### REQ-005 마이페이지 — 프로필 수정

**분류:** 회원관리 | **우선순위:** 중

로그인한 사용자가 본인 프로필을 수정한다.

- 수정 가능 항목: 이름, 연락처
- 이메일은 읽기 전용
- 저장 성공 시 "프로필이 업데이트되었습니다." 토스트 표시

---

### REQ-006 제품 목록 페이지

**분류:** 제품소개 | **우선순위:** 상

등록된 제품을 카드 그리드로 나열한다.

- 비로그인 사용자도 조회 가능
- 각 카드: 이미지(없을 경우 기본 아이콘), 제품명, 카테고리 배지, 요약
- 추천 제품(`is_featured`)에 "추천" 배지 표시
- 로딩 중 스켈레톤 UI 제공

---

### REQ-007 제품 상세 페이지

**분류:** 제품소개 | **우선순위:** 상

제품 slug를 URL로 사용하는 개별 제품 상세 페이지를 제공한다.

- 대표 이미지, 카테고리, 제품명, 요약, 상세 설명 표시
- "이 제품 문의하기" 버튼 → 온라인 문의 폼 이동
- 존재하지 않는 slug 접근 시 오류 안내 표시

---

### REQ-008 온라인 문의 폼

**분류:** 온라인문의 | **우선순위:** 상

비회원·회원 모두 문의를 접수할 수 있다.

- 로그인 사용자: 이름·이메일·연락처 자동 채움
- 비회원: 이름·이메일·연락처 직접 입력
- 문의 유형 선택: 제품문의 / 견적요청 / 기타
- 필수 항목(제목·내용) 미입력 시 유효성 오류 메시지 표시
- 제출 성공 시 DB 저장 후 완료 페이지로 이동

---

### REQ-009 문의 접수 완료 페이지

**분류:** 온라인문의 | **우선순위:** 중

문의 제출 성공 시 완료 안내를 표시한다.

- "24시간 이내 이메일 답변" 안내 메시지
- "홈으로" / "추가 문의" 이동 버튼

---

### REQ-010 마이페이지 — 내 문의 내역

**분류:** 온라인문의 | **우선순위:** 중

로그인한 사용자가 본인의 문의 목록과 답변을 조회한다.

- 표시 항목: 유형·상태 배지, 제목, 접수일
- 상태: 대기중 / 처리중 / 완료
- 관리자 답변이 있을 경우 문의 하단에 답변 내용 표시

---

### REQ-011 커뮤니티 게시판 목록

**분류:** 커뮤니티 게시판 | **우선순위:** 상

공지사항·자유게시판·Q&A 세 게시판을 탭으로 전환하여 조회한다.

- 비로그인 사용자도 목록·상세 조회 가능
- 데스크탑: 테이블 (번호·제목·작성자·날짜·조회수)
- 모바일: 카드 리스트
- 공지 글(`is_pinned`) 상단 고정 + "공지" 배지 + 배경색 강조

---

### REQ-012 게시글 작성 / 수정 / 삭제

**분류:** 커뮤니티 게시판 | **우선순위:** 상

로그인한 사용자가 게시글을 CRUD할 수 있다.

- 글쓰기: 게시판 선택(공지는 관리자 전용), 제목, 내용 입력
- 수정·삭제: 본인 글 또는 관리자만 가능
- 삭제 전 확인 다이얼로그 표시
- 수정 완료 후 게시글 상세 페이지로 이동

---

### REQ-013 관리자 — 문의 답변 관리

**분류:** 관리자 페이지 | **우선순위:** 중

관리자가 접수된 문의 목록을 조회하고 답변을 등록한다.

- 전체 문의 목록 (유형·제목·작성자·상태·접수일)
- 문의 클릭 시 원문 확인 + 답변 입력 모달 표시
- "처리중으로" 버튼과 "답변 완료" 버튼 분리 제공
- 답변 완료 시 상태 "resolved", 답변 일시 자동 기록

---

### REQ-014 관리자 — 제품 등록 / 수정 / 삭제

**분류:** 관리자 페이지 | **우선순위:** 중

관리자가 제품 정보를 관리한다.

- 제품 목록 테이블 (제품명·카테고리·추천 여부)
- 추가/수정: 모달에서 제품명·slug·요약·카테고리·추천 여부 입력
- slug는 영문 소문자·숫자·하이픈만 허용
- 삭제 전 확인 다이얼로그 표시

---

### REQ-015 관리자 — 회원 권한 관리

**분류:** 관리자 페이지 | **우선순위:** 하

관리자가 전체 회원 목록을 조회하고 권한을 변경한다.

- 목록 표시: 이름·이메일·연락처·권한·가입일
- 권한 전환: "일반 회원 ↔ 관리자" 즉시 반영
